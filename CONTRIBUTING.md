# Contributing

Contributions are welcome, especially body/firmware test reports and property mapping corrections.

## Body Test Reports

Please include:

- camera body
- firmware version
- USB mode used
- host OS and tool/library used
- whether slot reads worked
- whether property writes worked
- whether preset name writes worked
- any failed property codes or response codes

## Property Corrections

For property mapping corrections, include:

- property code
- old interpretation
- corrected interpretation
- test method
- sample values
- camera body and firmware

## Style

- Use cautious language: observed, tested, known, unknown.
- Avoid claiming official support.
- Separate confirmed behavior from guesses.
- Prefer small focused pull requests.
